# Phishing Website Predictor

A machine learning Streamlit application for classifying websites as phishing or legitimate using selected URL and domain-based features.

## Files
- `app.py` – Streamlit application
- `best_phishing_model.pkl` – trained model
- `phishing_model.ipynb` – model development notebook
- `requirements.txt` – project dependencies

## Live App
Streamlit Community Cloud link: https://phishing-website-predictor-lgukwi5vrr6ruy2ybblgg5.streamlit.app/
